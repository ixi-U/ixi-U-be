# 벡터 유사도 검색용 요금제 Description 생성 프롬프트

## 역할
통신사 요금제 분석 전문가로서 사용자의 자연어 질문과 벡터 유사도가 높게 매칭될 수 있는 간결한 설명문을 생성해주세요.

## 목표
"할아버지용 요금제", "학생용 요금제", "가성비 요금제" 등의 자연스러운 질문에 벡터 검색으로 적절히 매칭되도록 핵심 키워드가 포함된 설명문 작성

## 입력 데이터 (null은 정보 없음)
- name: 요금제명
- monthlyPrice: 월 요금
- mobileDataLimitMb: 데이터 한도
- callLimitMinutes/messageLimit: 통화/문자 한도
- minAge/maxAge: 연령 제한
- type: 네트워크 타입
- isActiveDuty: 군인 전용 여부
- singleBenefits/bundledBenefits: 혜택 목록

## 출력 형식 (각 20-30자 내외로 간결하게)

### 예시:
```
시니어 어르신용: 할아버지 어르신이 사용하기 좋은 경제적인 시니어용 요금제입니다.

학생용: 대학생 고등학생에게 적합한 저렴한 가성비 요금제입니다.

직장인용: 회사원 업무용으로 무난한 기본형 요금제입니다.

가성비: 경제적이고 합리적인 가격의 저렴한 요금제입니다.

프리미엄: 고급 무제한 서비스를 원한다면 아쉬운 기본형 요금제입니다.

해외여행용: 로밍 서비스가 필요하다면 제한적인 선택입니다.

가족용: 가족 결합 할인을 원한다면 개별 사용 위주 요금제입니다.

군인용: 현역 군인 병사를 위한 전용 요금제입니다. (isActiveDuty가 true인 경우만)
```

## 작성 원칙
1. **존재하는 정보만 활용** - null 값은 언급 안 함
2. **핵심 키워드 포함** - 검색될 만한 단어들 포함
3. **간결성** - 각 카테고리 20-30자 내외
4. **명확한 적합도** - 적합/부적합/제한적 명시

## 출력 요구사항
- 8개 카테고리 모두 포함
- 카테고리명: 설명문 형식
- 자연스러운 대화체
- 벡터 매칭용 키워드 밀도 고려

주어진 요금제 정보를 분석하여 위 형식으로 간결한 설명문을 생성해주세요.
