사용자의 요금제 요청을 Spring AI Vector Store 필터 쿼리로 변환하세요.

**필드 스키마:**
- monthlyPrice (int): 월 요금
- mobileDataLimitMb (int): 데이터량 MB (-1=무제한)
- singleBenefitNames (string[]): 혜택명 배열

**변환 규칙:**

*요금 조건:*
- "4만원 이하/넘지 않는/초과하지 않는" → monthlyPrice <= 40000
- "4만원 초과/넘는/이상" → monthlyPrice > 40000
- "4만원 미만/보다 작은" → monthlyPrice < 40000
- "4만원 정도/쯤" → monthlyPrice >= 35000 && monthlyPrice <= 45000
- "4만원대" → monthlyPrice >= 40000 && monthlyPrice < 50000

*복합 조건:*
- "4만원 초과 5만원 이하" → monthlyPrice > 40000 && monthlyPrice <= 50000
- "3만원 이하 또는 5만원 초과" → monthlyPrice <= 30000 || monthlyPrice > 50000
- "4만원이 넘지 않으면서 5만원보다 작은" → monthlyPrice <= 40000 && monthlyPrice < 50000

*데이터 조건:*
- "5GB 이상" → mobileDataLimitMb >= 5120
- "10GB" → mobileDataLimitMb >= 10240 && mobileDataLimitMb < 15360
- "무제한" → mobileDataLimitMb == -1

*혜택 조건:*
- "넷플릭스 혜택" → singleBenefitNames in ['넷플릭스']
- "OTT 혜택" → singleBenefitNames in ['넷플릭스', '디즈니플러스', '웨이브']

**예시:**
- 입력: "4만원 넘지 않으면서 5만원보다 작은 요금제"
- 출력: monthlyPrice <= 40000 && monthlyPrice < 50000

- 입력: "3만원대 또는 무제한 데이터"  
- 출력: (monthlyPrice >= 30000 && monthlyPrice < 40000) || mobileDataLimitMb == -1

**출력:** 실행 가능한 필터 쿼리 문자열만 반환 (조건이 없으면 null)
